```Node.js npm install ght```

```Require.js require(["ght"], ...```

```Bower bower install ght```

```Angular.js angular.module('yourApp', ['ght'])...```
#Examples

encode:
```javascript
ght.encode('دوست دارم')
```
return:
```html
j4od jal2
```

decode:
```javascript
ght.decode('j4od jal2')
```

return:
```html
دوست دارم
```
