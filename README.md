#Installation

```Node.js npm install gt```

```Require.js require(["gt"], ...```

```Bower bower install gt```

#Examples

encode:
```javascript
gt.encoder('دوست دارم')
```
return:
```html
j4od jal2
```

decode:
```javascript
gt.decoder('j4od jal2')
```

return:
```html
دوست دارم
```
