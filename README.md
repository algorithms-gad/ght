```Node.js npm install ght```

```Require.js require(["ght"], ...```

```Bower bower install ght```

#Examples

encode:
```javascript
ght.encoder('دوست دارم')
```
return:
```html
j4od jal2
```

decode:
```javascript
ght.decoder('j4od jal2')
```

return:
```html
دوست دارم
```
